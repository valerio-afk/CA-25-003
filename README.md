# Introduzione

Questo repository contiene il codice sorgente del mio programma che implementa l'algoritmo di **Gilbert–Johnson–Keerthi** .

Il codice presente in questo repository è stato scritto durante le riprese del video [**CA-25-003a**](https://youtu.be/gu_taxfiV2I) del canale YouTube [AFK](https://www.youtube.it/@valerio_afk).
